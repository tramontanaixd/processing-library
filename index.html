Tramontana for Processing

A Library by Pierluigi Dalla Rosa for the Processing programming environment.
Last update, 5th July 2017.

Tramontana - prototyping and creative kit for multi-device interactions and experiences.
With Tramontana you can access the inputs and outputs of one or more smartphones (or other devices) to easily and quickly prototype interactive spaces, connected products or whatever you can think of. What used to involve complex tasks like networking, native app development, etc. can now be created with a single sketch on your computer.
Feel free to replace this paragraph with a description of the Library.
Contributed Libraries are developed, documented, and maintained by members of the Processing community. Further directions are included with each Library. For feedback and support, please post to the Discourse. We strongly encourage all Libraries to be open source, but not all of them are.

Download

Download Tramontana Processing Library version 1.0.5 (13) in .zip format.

Installation

Unzip and put the extracted Tramontana folder into the libraries folder of your Processing sketches. Reference and examples are included in the Tramontana folder.

Keywords. interactive spaces, interactive environment, interaction design prototyping
